# API Wrapper

* Setup

** Create a virtual environment, acivate it. For example:
*** conda create -n apimapper python=3.6
*** conda activate apimapper 
** pip install -r requirements.txt

* Install package

** from PyPi: 
pip install apimapper
** from source: 
pip install -e .

* Unit Testing
tests> python -m pytest
